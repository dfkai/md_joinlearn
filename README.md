# Markdown 练习器

一个帮助你快速掌握 Markdown 语法的在线工具。

**joinlearn.com 出品**

## 功能

- **练习模式**：自由输入，实时预览
- **挑战模式**：15 关闯关练习，巩固语法
- **深色模式**：支持明暗主题切换
- **语法工具栏**：一键插入常用语法

## 本地运行

```bash
npm install
npm run dev
```

访问 http://localhost:3000

## 部署

直接推送到 GitHub，然后在 [Vercel](https://vercel.com) 导入仓库即可自动部署。

## 技术栈

- Next.js 15
- React 19
- Tailwind CSS
- TypeScript
